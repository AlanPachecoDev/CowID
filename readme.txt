Para generar apk con expo

1. Primero registrare en expo.

2. Luego correo expo login y logueate desde la terminal del proyecto.

3. Configura tu proyecto para EAS Build. Puedes hacer esto ejecutando eas init en 
   el directorio raíz de tu proyecto. Debes seguir las instrucciones para configurar tu proyecto correctamente.
	
   Para ello debes tener instalado en tu equipo (npm install -g eas-cli)

4. Ejecuta el comando eas build --platform android para iniciar el proceso de construcción de tu aplicación. 
Este comando generará un archivo APK que podrás descargar desde la plataforma EAS.



expo build:android
